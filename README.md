# شركة الراية للبرمجة والتصميم - الموقع الرسمي

هذا هو الموقع الرسمي لشركة الراية للبرمجة والتصميم، تم تطويره باستخدام HTML و Tailwind CSS ليقدم تجربة مستخدم عصرية واحترافية.

## المميزات الجديدة
- **صفحات تفصيلية:** تم إضافة `service.html` و `blog.html` كقوالب للتفاصيل.
- **عرض الأعمال:** تم توسيع قسم الأعمال ليشمل مشاريع متنوعة (طبية، تعليمية، بيئية، إلخ).
- **باقات الأسعار:** تنظيم الأسعار في فئات (مواقع، هوية، تطبيقات) ليسهل على العميل الاختيار.
- **الهوية البصرية:** دمج شعار الشركة في الهيدر والفوتر.

## تعليمات الرفع على GitHub

لرفع هذا المشروع على حسابك في GitHub، اتبع الخطوات التالية:

1. قم بإنشاء مستودع (Repository) جديد على GitHub باسم `al-rayaa-website`.
2. افتح نافذة الأوامر (Terminal) في مجلد المشروع وقم بتنفيذ الأوامر التالية:

```bash
# إضافة جميع الملفات
git add .

# تسجيل التغييرات
git commit -m "الإطلاق الأول للموقع مع التحسينات الجديدة"

# ربط المستودع المحلي بالمستودع على GitHub (استبدل USERNAME باسم مستخدمك)
git remote add origin https://github.com/USERNAME/al-rayaa-website.git

# رفع الملفات
git branch -M main
git push -u origin main
```
